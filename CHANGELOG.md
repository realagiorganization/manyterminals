# Changelog

All notable changes to this project are documented in this file.

The format is based on Keep a Changelog and versions are tracked with lightweight semantic tags.

## [Unreleased]

### Added

- `record-fixture` command for capturing live terminal inspections into reusable regression fixtures.

## [0.1.0] - 2026-03-29

### Added

- Initial Codex skill scaffold for coordinating many Linux terminal emulators.
- Terminal inspection, tmux planning, empty-terminal closing, and `gh` publishing commands.
- Unit, CLI, tmux integration, BDD, Docker, and GitHub Actions coverage.
- Recorded UI demo, GitHub Pages publishing, MIT licensing, and release automation.
